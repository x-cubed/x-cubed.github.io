Growl for Android

A Growl client/server designed for Android devices, to allow them to receive Growl notifications from other devices on the same network.

This software is available on the Google Play Store.

You can see our plans in the Feature List.

Use of the name and logo of Growl has been approved by The Growl Project.

Getting Started

You'll need to have a computer running Growl for Windows, or a build of Growl for Mac that has support for GNTP.

  • Subscribe to notifications from Growl for Windows
  • Forward notifications from Growl for Windows
